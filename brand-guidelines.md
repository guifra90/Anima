# Mirror Agency — Brand Guidelines

### Brand Personality
- Sophisticated
- Intelligent
- Minimalist
- Future-oriented

### Color Palette
- **Mirror Black**: #0A0A0A (Dark Mode Base)
- **Fluid Silver**: #E5E7EB (Accents)
- **Deep Slate**: #1F2937 (Surface Colors)
- **Neon Cyan**: #22D3EE (Active Indicators / Highlights)

### Typography
- Header: **Inter** or **Outfit**
- Body: **Inter**
- Code: **JetBrains Mono** or **Fira Code**

### Design Principles
- **Gradients**: Use smooth, subtle gradients of dark greys and slates.
- **Glassmorphism**: Use backdrop-blur for dashboard components.
- **Micro-animations**: Use Framer Motion for smooth transitions between agent states.
- **Visual Feedback**: Every autonomous action should have a clear, pulsing visual indicator in the UI.
